Syllabus
The Students will be able
To explain in detail about the Automation Anywhere Enterprise Platform, Architecture, and it’s Components.
To provide in-depth knowledgeabout the various features and functionalities of AA software, Runtime AA Client, and Web Control Room.
To create Bots using different types of Recorders, Editors, and Basic Commands using Automation Anywhere platform.
To apply bot creation methodologies for developing bots to solve any real world problems.
The students will be able to
Automate any sort of manual or repetitive human tasks by using the Bot Development tool.
Develop and deploy software bots which can perform automation tasks 24/7 without interruption.
Create and deploy bots which significantly improves the process and productivity of any industrial, educational, governmentand private organizations.
Build IQ bots with artificial intelligence which are capable of solving complex real world problems.
Mohamed Sathak A J College of Engineering has partnered with Automation Anywhere the market leaders in Robotic
Process Automation (RPA) to establish the AAU powered Bot Lab.
The AAU powered Bot Labprovides training in Robotic Process Automation (RPA) which is simpleand powerfulautomation software.
RPAis used to create software robots that can
automate any business process.
Access to the Automation Anywhere Enterprise Platform and hands-on training.
Globally-recognized certification, internships and placement assistance across industries.
Opportunity to build your own bots and upload it on Automation Anywhere Bot Store, the world's leading marketplace for bots and Digital Workers.
